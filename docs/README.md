# Documentation

This directory contains additional documentation and assets for the MaixCode extension.

## Structure

- `images/` - Screenshots and diagrams
- `examples/` - Extended examples and tutorials
- `api/` - API documentation
- `development/` - Development guides

## Screenshots Needed

- Device discovery panel
- Image streaming viewer
- Code debugging session
- Example browser
- Configuration settings

## Adding Screenshots

To add screenshots:
1. Take screenshots in VS Code with MaixCode active
2. Save in `images/` directory with descriptive names
3. Update README.md with proper image references
4. Ensure images are optimized for web viewing
